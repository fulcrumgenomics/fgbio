# `fgbio` - Fulcrum Genomics Bioinformatic Tools

## Building `fgbio` 

Use ```sbt assembly``` to build an executable jar in ```target/scala-2.11/```.  
Tests may be run with ```sbt test```.

## Java Requirements
Java SE 8 is required.
